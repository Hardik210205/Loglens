# LogLens Frontend

This folder contains a React + TypeScript frontend created with Vite.

## Getting started

```
npm install
npm run dev
```

The app runs on `http://localhost:3000` by default.

### Features

- Vite build with React + TS
- Pages: Live Logs, Incident Explorer, Dashboard
- React Query for API interaction
- SignalR client for live log streaming
- Recharts used in dashboard
- Modular component structure

No authentication is configured; backend API assumed at `/api`.
